# viglucci.github.io

Source code for [viglucci.io](https://viglucci.io)

## Development

```bash
npm run develop
```

## Deployment

### Install the now cli

```bash
$ npm install -g vercel
```

### Build & deploy to 'now'

```bash
$ vercel
```

### Update the DNS alias

```
$ vercel --target production
```
